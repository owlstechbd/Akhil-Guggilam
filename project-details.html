A Full-Stack Grocery Delivery Website

I developed this project as a self-initiated full-stack application to simulate a modern grocery delivery platform. Customers can browse groceries, add items to their cart, and place secure orders, while admins can manage products, update stock, and process orders from a dashboard. The system is responsive and designed for scalability, ensuring smooth experiences on both desktop and mobile.

  • Frontend: React.js, CSS (Responsive UI), Stripe.js
  • Backend: Node.js, Express
  • Database: MongoDB
  • Other Tools: REST APIs, Stripe (Payment Intents + Webhooks), JWT auth
